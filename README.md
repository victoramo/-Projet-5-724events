🟩 Projet 5 — 724events


📝 Description
Campagne de tests exploratoires sur l'application événementielle 724events. Ce projet met l'accent sur la structuration des tests, leur traçabilité et la production d'un reporting exploitable.

🎯 Objectif
Structurer les tests exploratoires, assurer leur traçabilité via un outil de gestion, et produire un reporting clair à destination de l'équipe.

🛠️ Travail réalisé
Conception et exécution de tests exploratoires

Suivi des anomalies et des cas de test dans Jira

Rédaction d'un bilan de campagne de tests complet
